
Projeto Fatec: TechVagas

Projeto desenvolvido para as disciplinas de Engenharia de Software e Web II onde foi desenvolvido a parte de login, cadastro do candidato sendo este estudante ativo ou exaluno da Fatec Franca.

O projeto foi desenvolvido com backend em NodeJs com Express, Cors, Bcrypt com integração com o frontend que foi desenvolvido em HTML, CSS e JavaScript.
